# Superprint Information System

University course project focused on information system design and analysis for a real business case.

This repository contains an academic team project developed as part of university coursework in information systems design. The project is based on the analysis and modeling of business processes for Superprint d.o.o.

## Project scope
The project includes:
- business system analysis
- decomposition diagrams
- PK matrix
- BPMN model
- use cases
- ERA model
- relational schema
- Oracle APEX application documentation

## My contribution
My role in this project was focused on the Oracle APEX implementation. I worked on building forms and master-detail forms used for entering, organizing, and reviewing data inside the application, as well as report pages for displaying records in a structured way.
